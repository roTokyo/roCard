Privacy policy

Considero la privacy dei miei visitatori estremamente importante e questa informativa delinea i tipi di informazioni personali che vengono ricevute e raccolte e il modo in cui vengono utilizzate.

Come vengono gestiti i tuoi dati

Questo sito web è sviluppato con Jekyll, ed è ospitato su GitHub.

Questo sito

Come in molti altri siti Web, utilizzo i file di registro per capire quando, da dove e con quale frequenza scorre il traffico verso questo sito. Le informazioni contenute in questi file di registro includono:

  • Indirizzi IP
  • Tipi di browser
  • Provider di servizi Internet
  • Data e ora
  • Pagine visitate e pagina d'uscita
  • Numero di clic

Tutte queste informazioni non sono collegate a nulla che sia personalmente identificabile.

Cookies

Questo sito utilizza un cookie denominato RobertoCalesiniCookiesAccepted per attivare il caricamento di CSS nella cache nei futuri carichi di pagina e per tenere traccia dell'accettazione delle condizioni d'uso e della politica di privacy da parte dell'utilizzatore del sito.

Quando visiti questo sito anche alcuni cookies di terze parti vengono memorizzati sul tuo computer.

Se si desidera disabilitare i cookie, è possibile farlo attraverso le opzioni del proprio browser web. Le istruzioni per farlo possono essere trovate sui siti Web specifici dei browser web.

Github

Github, come qualsiasi azienda di hosting, può avere una visione completa dei dati dei visitatori, come la connessione di indirizzi IP, pagine visitate, ecc. Si noti che Github non è noto per la profilazione attiva dei visitatori. Utilizzando una VPN puoi (provare a) impedirlo.

Per condividere commenti o fornire indicazioni sul sito, puoi utilizzare anche Github, il mio sistema preferito perché questa societa non è nota per la profilazione attiva dei visitatori. Github ha inoltre una reputazione decente ed è possibile ottenere un accordo di elaborazione dei dati (DPA) che però non ho ancora sottoscritto. Con tale accordo non verrà inviata alcuna informazione aggiuntiva a terzi (poiché tutte le richieste sono già state inviate a GitHub).

Google Analytics

Questo sito web utilizza Google Analytics. Ciò accade solo se approvi cookie e script di terze parti. Google Analytics è uno strumento di analisi dei dati web che utilizzo per capire in che modo i visitatori interagiscono con questo sito Web e segnala le tendenze dei visitatori utilizzando cookie e web beacon senza identificare i singoli visitatori. Leggi Informativa sulla privacy di Google.

Crisp Chat

Il modulo di chat nell'angolo in basso a destra utilizza Crisp Chat e al caricamento di quel modulo, i dati dei visitatori vengono inviati a Crisp Chat. Ciò accade solo se approvi cookie e script di terze parti. Leggi Informativa sulla privacy di Crisp Chat.

Ho attivato un moderno servizio di chat per poter comunicare in tempo reale con chi mi legge. Anche questo è un servizio gestito da terzi perciò se non ti senti a tuo agio nell'utilizzarlo non utilizzare questo servizio e contatttami via mail all'indirizzo rotokyo@icloud.com.
